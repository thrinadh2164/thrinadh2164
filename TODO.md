# 🚀 Profile Expansion Roadmap

This document outlines future enhancements to continue building the "Cyberpunk Fire" brand across the portfolio.

## 1. GitHub Profile Optimization
- [ ] **Pin Top Repositories:** Go to the main GitHub profile page and manually pin 4-6 of your most impressive technical projects (e.g., Gen AI, RAG pipelines, or full-stack apps).
- [ ] **Update COLLAB.md:** Write a themed markdown file in `docs/COLLAB.md` explaining your preferred tech stack for open-source work and how to reach out for freelance/collaboration opportunities.

## 2. Dedicated Projects Website
- [ ] **Create `My_Projects` Repo:** Similar to `My_Awards` and `My_Certifications`, create a dedicated GitHub Pages site for projects.
- [ ] **Showcase Details:** Include high-quality screenshots, system architectures, and live demo links for each major project.
- [ ] **Update SVG Link:** Change the link in `README.md` so that `panel_projects.svg` points to the new `My_Projects` URL instead of the default repositories tab.

## 3. Routine Maintenance
- [ ] **Monitor API Widgets:** Occasionally check that the `github-readme-activity-graph` and `github-readme-streak-stats` are rendering properly (free APIs occasionally experience downtime).
- [ ] **Add New Tech:** As you learn new frameworks, add their devicon SVGs to the Technologies grid in `README.md`.
